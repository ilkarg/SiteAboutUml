UML - это...

UML (Unified Modeling Language) - это графический язык моделирования общего назначения, предназначенный для спецификации,
визуализации, проектирования и документирования всех артефактов, создаваемых в ходе разработки.

Для разработки Web-приложений UML располагает специальным расширением Web Application Extension (WAE), которое дополняет
систему обозначений новыми элементами, позволяющими разработчикам моделировать специфические архитектурные элементы Web-приложений.
Это – подмножество диаграмм классов для Web, дающее возможность адекватно и наглядно отображать описанную выше специфику обработки данных в сети.
